# TravelGenie-AI-Powered-Personalized-Travel-Itinerary-Planner-via-Telegram
TravelGenie is an intelligent AI travel planner integrated with Telegram that creates personalized trip itineraries based on user input. Utilizing advanced GPT-4 AI models and n8n workflows, it delivers real-time, context-aware travel recommendations directly to users, making trip planning effortless and interactive.
